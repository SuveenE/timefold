# Timefold

Simple desktop app built with Electron, React, and TypeScript.

## Prerequisites

- Node.js 18+
- npm 9+

## Run locally

```bash
npm install
npm start
```

This starts webpack dev servers and opens the Electron window.

## Checks

```bash
npm run lint
npm test -- --runInBand
```

## Build production bundles

```bash
npm run build
```
